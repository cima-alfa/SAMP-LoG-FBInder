# Keybinder for Life of German SAMP

Built using AHK v1.1 32-bit ASI

## Features:

- Keybinds, loops and commands with multi line text support
- Spam protection to prevent command spam ban
- Custom variables usable in texts
- Ability to turn on / off keybinder without shutting down the application
- Ability to stop active loops (including multi line texts)
- AFK Timer
- Auto update with release notes
- Settings saved in a JSON file
- Overlay
- LoG specific positions
- SAMP UDF Integration
- Support for SAMP 0.3.7 R5
- Dev mode to test your keybinds and commands without sending anything to the server
- Quick weapon switching - bind specific weapon classes
- Automatic traffic stop with detection of player vehicle, necessary license type, rest time check for LKWs and automatic confiscation of illegal items
- Privacy focused, no need to read chat logs and other documents to interact with chat actions

### Antivirus

- It may be necessary to add an antivirus exception to the directory in which the keybinder is located.

## Installation

- Download the keybinder.exe from the [official release page](https://github.com/cima-alfa/SAMP-LoG-FBInder/releases/latest)
- Optionally download the example keybinder_settings.json so you don't have to configure everyting from scratch
- Add the downloaded files to a separate directory
- It will most likely be necessary to add an antivirus exception to the keybinder directory
